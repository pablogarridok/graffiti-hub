# Blog Graffiti - MVC PHP

Sistema de blog con arquitectura MVC en PHP, MySQL y Docker.

##  Instalación

### Requisitos previos
- Docker
- Docker Compose

### Pasos

1. **Clonar el repositorio**
```bash
git clone <tu-repo>
cd proyecto-blog
```

2. **Levantar los contenedores**
```bash
docker-compose up -d
```

3. **Acceder a la aplicación**
- Blog: http://localhost:8080
- phpMyAdmin: http://localhost:8081
